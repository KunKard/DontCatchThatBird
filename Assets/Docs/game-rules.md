# 《Don't Catch That Bird》游戏规则

> 版本：v0.3 | 更新：2026-07-02

---

## 一、基础规则

### 1.1 游戏目标

抓住键盘上跳跃的小鸟以获得分数。避开危险键（DangerKey）。分数越高越好。

### 1.2 游戏结束条件

| 条件 | 说明 |
|---|---|
| 按下 DangerKey | 立即 Game Over |
| Miss 计数达到 5 | 小鸟飞走，Game Over |

### 1.3 按键判定

| 按键 | 结果 |
|---|---|
| 按鸟所在的键 | ✅ 抓取成功 |
| 按 DangerKey | ❌ Game Over |
| 按其他键 | ⛔ Combo 清零，Miss +1 |

---

## 二、小鸟机制

### 2.1 小鸟行为

- 小鸟随机停留在虚拟键盘 37 个键中的某一个
- 每隔一段时间自动跳到新键
- 跳跃时 **不计入 Miss**（普通跳跃 +1 Miss）
- **小鸟可以跳到 DangerKey 上**：此时不计 Miss，但按该键即 Game Over

### 2.2 跳跃间隔（难度曲线）

| Score | 跳跃间隔 |
|---|---|
| 0~49 | 2.0 ~ 2.5s |
| 50~99 | 1.5 ~ 2.0s |
| 100~199 | 1.0 ~ 1.5s |
| 200+ | 0.6 ~ 1.0s |

可在 GameConfigSO 中自定义。

---

## 三、计分与 Combo

### 3.1 计分公式

```
每次抓取得分 = 10 + Combo × 5
```

### 3.2 Combo 系统

| Combo | 文字 | 颜色 | 字号 |
|---|---|---|---|
| 0~4 | 连击: N | 黑色 | 28 |
| 5~9 | 连击: N! | 蓝色 `#007AFF` | 34 |
| 10~14 | 连击: N!! | 橙色 `#FF8000` | 36 |
| 15+ | 连击: N!!! | 红色 `#FF3F00` | 38 |

- Combo 连续抓取递增
- 按错键 Combo 清零
- **无敌状态下按错键不扣 Combo**

### 3.3 分数飘出效果

| 得分 | 字号 |
|---|---|
| < 25 | 24 |
| 25~49 | 28 |
| 50~74 | 32 |
| ≥ 75 | 36 |

---

## 四、DangerKey

### 4.1 基本规则

- 每只鸟绑定一组 DangerKey（鸟存活期间不变）
- DangerKey 数量随分数递增
- **优先出现在小鸟物理键盘位置的四周**（70% 概率）

### 4.2 DangerKey 数量

| Score | 数量 |
|---|---|
| 0~99 | 1 个 |
| 100~199 | 2 个 |
| 200+ | 3 个 |

---

## 五、Miss 机制

### 5.1 Miss 来源

| 行为 | Miss 变化 |
|---|---|
| 鸟自动跳跃 | +1 |
| 按错键 | +1 |
| 鸟跳到 DangerKey | **不计** |

### 5.2 Miss 上限

- 默认上限：**5**
- Miss ≥ 5 → GameOver
- **无敌状态下所有 Miss 来源均无效**
- Miss ≥ 4 时文字变红色警告

---

## 六、道具系统

### 6.1 出现规则

- 每抓取 **8~12 只**小鸟触发一次道具
- 循环触发，每轮 8~12 只可选触发
- 道具出现在随机键上（不与鸟键/DangerKey 重合）
- 道具持续时间随分数递减

### 6.2 道具类型

| 道具 | 图标 | 效果 | 持续时间 |
|---|---|---|---|
| ❄️ 冻结 | 雪花 | 鸟静止不动，Miss 暂停 | 5 秒 |
| 💗 无敌 | 心形 | Miss 不增加，DangerKey 不致死，Combo 不扣 | 4 秒 |

### 6.3 道具持续时间

| Score | 持续时间 |
|---|---|
| < 200 | 2.5s |
| 200~999 | 2.0s |
| 1000~4999 | 1.5s |
| ≥ 5000 | 1.0s |

### 6.4 道具视觉反馈

| 道具 | 遮罩颜色 | 粒子颜色 |
|---|---|---|
| 冻结 | 淡蓝色半透明 | 冰蓝 |
| 无敌 | 淡绿色半透明 | 淡绿 |

---

## 七、金鸟

### 7.1 触发条件

Combo 达到 **50 的倍数**（50 / 100 / 150 / ...）时，下一只生成的鸟为金色。

### 7.2 奖励

- 抓到金鸟：**得分 ×2**
- 粒子效果：20 个金色方块华丽爆出

---

## 八、双鸟模式

### 8.1 触发条件

Score > 1000 时，**10% 概率**出现两只小鸟。

### 8.2 规则

- 两只鸟出现在不同键位
- 双鸟跳跃间隔 **+0.5s**（比正常慢）
- 必须在 **0.2 秒内同时抓到**两只 → **得分 ×3**
- 超时未同时抓到 → 第二只飞走，正常得分
- 双鸟期间第一只被抓后**不立即刷新新鸟**，等两只都抓到或超时才刷新

## 九、视觉效果

| 事件 | 效果 |
|---|---|
| 抓取成功 | 鸟弹跳 + 8 个彩色粒子爆出 + 分数飘出 |
| 按错键 | 按键浅红闪 + 低沉音效 |
| 按 DangerKey | 按键深红闪 + 警报音 + **全屏红闪 0.5s** |
| DangerKey 变更 | 文字闪黄 0.2s |
| 金鸟抓取 | 20 个金色粒子 + 2 倍分 |
| 双鸟同时捕获 | 金色粒子 + 3 倍分 |
| 道具拾取 | 对应颜色粒子爆出 + 半透明遮罩 |
| 破纪录 | 庆祝音效 + 绿色 `[新纪录！]` |
| Miss ≥ 4 | 文字变红预警 |

## 十、操作速查

| 操作 | 效果 |
|---|---|
| 按鸟键 | 抓取 + 得分 |
| 按 DangerKey | Game Over |
| 按其他键 | Combo 清零 + Miss +1 |
| 按道具键 | 拾取道具 |
| 空格（GameOver 后） | 重新开始 |
| 空格（Ready 界面） | 开始游戏 |
| 按住 Alt | 显示鼠标 |

## 十一、界面速查

| 位置 | 元素 |
|---|---|
| 左上 | 得分 / 连击 |
| 右上 | 危险键 / 失误数 |
| 顶部居中 | 最高分 |
| 键盘上方 | 小鸟下次跳跃倒计时 |
| 屏幕下方 | 树枝（多根，鸟停留） |
