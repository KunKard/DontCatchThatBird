## 一、已实现功能

### 核心玩法

| 功能 | 说明 |
|---|---|
| 虚拟键盘 | 37 键 QWERTY 布局（5 行：数字/字母/空格），TMP 中文手绘字体显示 |
| 小鸟自动跳跃 | 1.0~2.5s 随机间隔，分数越高跳越快（分 4 个难度段） |
| 物理键盘输入 | `Input.GetKeyDown` 监听 37 键，正确键抓取 / DangerKey 死亡 / 错误键 Combo 清零 + Miss++ |
| DangerKey | 与鸟绑定，数量随分数递增（1→2→3），优先出现在小鸟物理键盘位置的四周（曼哈顿距离 ≤2） |
| 多 DangerKey | Score≥100 → 2 个，Score≥200 → 3 个 |
| 计分公式 | `Score += 10 + Combo × 5` |
| Combo 系统 | 累积递增，按错清零；视觉分级（蓝! / 橙!! / 红!!!） |
| Miss 机制 | 鸟跳跃 Miss++，按错键 Miss++，Miss≥5 → GameOver；鸟跳到 DangerKey 不计 Miss |
| 抓取粒子 | 8 个彩色方块爆出飞散 + 淡出（后续可替换为羽毛 Sprite） |
| DangerKey 全屏闪烁 | 按到 DangerKey 时全屏红色半透明闪烁 0.5s |
| GameOver | DangerKey 死亡 / Miss≥5 小鸟飞走，均有原因提示 |
| 空格重开 | GameOver 后按空格重新开始 |
| Best Score 持久化 | PlayerPrefs 保存最高分，破纪录绿色提示 `[新纪录！]` |

### UI 系统

| 功能 | 说明 |
|---|---|
| 文本 | 全中文（STHUPO 繁圆字体），含：得分 / 连击 / 危险 / 失误 / 下次跳跃 / 最高分 |
| Combo 特效 | 5~9 蓝色 + `!`，10~14 橙色 + `!!`，15+ 红色 + `!!!`，字号递增 |
| Combo 断裂 | 按错键 Combo 清零，文字恢复黑色小字 |
| DangerKey 闪烁 | 变更时文字闪黄 0.2s |
| Miss 预警 | Miss≥4 文字变红 |

### 音效

| 功能 | 说明 |
|---|---|
| 抓取音效 | 按对键时播放 |
| 错误音效 | 按错键时播放 |
| DangerKey 音效 | 按到 DangerKey 时播放 |
| 破纪录音效 | 刷新历史最高分时播放 |
| 跳跃音效 | 鸟每次跳跃时播放 |
| BGM | 拖入多首，启动时随机选一首循环播放 |

### 小鸟 & 树枝

| 功能 | 说明 |
|---|---|
| Ready 小鸟 | 启动时显示一只鸟在键盘上方 + 玩法提示，按空格后飞向键位开始游戏 |
| 三态 Sprite | 正常（键盘上）/ 惊讶飞起（被抓瞬间）/ 满足蹲坐（树枝上），已拖入美术资源 |
| 多根树枝 | Inspector 中拖入多根树枝 GameObject，鸟随机飞向其中一根 |

| 飞行过程可见 | 0.6s 从键盘飞到树枝，缩小至 80×80，落地切换站立 Sprite |
